  • git init - ініцюємо пустий репозиторій

  • git commit - коментар нашого коду

  • git checkout створити нову гілку в проекті
  • git branch - змінюваня назви основної вітки
  • git push - відпрявляємо
  • git pull - отримуємо
  • git merge - зливає мій код з з кодом проекту
  • git clone - клонує проект
  • .gitignore - файл, який містить список всіх вказаних файлів та папок проекту, які Git повинен ігнорувати та не відстежувати.
